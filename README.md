# Orientación a Objetos 1

Este repositorio contiene ejercicios y trabajos prácticos realizados para la materia
**Orientación a Objetos 1** de segundo año, cuyo objetivo es introducir y consolidar los
conceptos fundamentales del paradigma de programación orientada a objetos.

## Contenidos

Introducción al paradigma de la Orientación a Objetos: clases, objetos,
encapsulamiento, herencia y polimorfismo. Modelado del dominio y diseño de
software mantenible y reutilizable, buenas prácticas, UML, principios SOLID
y testing unitario.
